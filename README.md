## 📋 项目简介

这是一个精心整理的加密货币工具导航站，收录了交易所、数据分析、钱包、DeFi、挖矿等多个领域的优质工具，帮助加密货币用户快速找到需要的服务。

## ✨ 主要特性

- 🎯 **分类清晰** - 10大工具分类，覆盖加密货币生态各个环节
- 🔍 **实时搜索** - 支持关键词快速检索工具
- 📱 **响应式设计** - 完美适配手机、平板、电脑
- 🎨 **现代化UI** - 简洁美观的卡片式布局

## 🚀 快速开始

### 方法1：直接使用
```bash
# 克隆仓库
git clone https://github.com/weinotes/crypto-tools-nav.git

# 打开HTML文件
cd crypto-tools-nav
open crypto-tools-nav.html  # Mac
start crypto-tools-nav.html # Windows
```

### 方法2：启用GitHub Pages
1. 进入仓库的 `Settings` → `Pages`
2. 在 `Source` 中选择 `main` 分支
3. 保存后等待几分钟
4. 访问 `https://weinotes.github.io/crypto-tools-nav/`

### 方法3：部署到其他平台
- **Vercel**: 导入GitHub仓库，自动部署
- **Netlify**: 拖拽HTML文件，即时上线
- **Cloudflare Pages**: 连接仓库，一键部署

## 📝 如何添加新工具

在 `crypto-tools-nav.html` 中找到对应分类，复制以下代码模板：

```html
<a href="工具网址" target="_blank" class="tool-link">
    <div class="tool-card">
        <div class="tool-header">
            <div class="tool-logo">🔷</div>
            <div class="tool-name">工具名称</div>
        </div>
        <div class="tool-description">工具描述</div>
        <div class="tool-tags">
            <span class="tag">标签1</span>
            <span class="tag">标签2</span>
        </div>
    </div>
</a>
```
